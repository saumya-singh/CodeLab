Why CodeGrader?

CodeGrader is an online judge which takes your code, compiles it, checks for compilation errors, execute your code by giving some hidden inputs and collecting the output your code gave, checks for runtime errors and finally matches your output with their hidden test outputs.



Features

1.Compile and execute code according to defined language
2.Sandboxing (using virtual environment)
3.Handling malicious programs
4.Retrieving data from hard disk for various test cases
5.Update results
6.Show the output or error messages for each submission.
